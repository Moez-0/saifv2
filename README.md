# Saifeddine Lahmar Portfolio

A modern React + Vite portfolio website for Saifeddine Lahmar, focused on film, journalism, and digital storytelling.

## Tech Stack

- React 19
- TypeScript
- Vite
- Tailwind CSS v4
- Motion

## Run Locally

**Prerequisites:** Node.js 20+

1. Install dependencies:
   `npm install`
2. Start development server:
   `npm run dev`
3. Build for production:
   `npm run build`
